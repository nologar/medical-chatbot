# medical-chatbot

Este proyecto es un prototipo de aplicación que conecta con una API alojada en un servicio cloud como AWS. Se trata de un chatbot clínico que permite realizar consultas médicas y devuelve, como respuesta, una recomendación sobre la necesidad o no de derivar a un paciente a cardiología. Además, la respuesta incluye los fragmentos más relevantes utilizados para generar dicha recomendación, así como la localización de las guías clínicas de las que se han extraído.
